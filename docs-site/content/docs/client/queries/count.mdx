---
title: Count
description: Count rows matching a filter condition using Drizzle aggregations.
icon: Hash
---

Use `db.select()` with the `count()` function to count rows.

```typescript
import { drizzle } from '@superapp/db'
import { count } from 'drizzle-orm'
import * as schema from './generated/schema'

const db = drizzle({ connection, token, schema })

const [result] = await db.select({ count: count() })
  .from(schema.orders)
// result.count: number
```

## Count with Filter

```typescript
import { count, eq } from 'drizzle-orm'

const [result] = await db.select({ count: count() })
  .from(schema.orders)
  .where(eq(schema.orders.status, 'active'))
// result.count: number
```

## Count with Complex Filters

```typescript
import { count, eq, gte, and } from 'drizzle-orm'

const [result] = await db.select({ count: count() })
  .from(schema.orders)
  .where(and(
    eq(schema.orders.status, 'active'),
    gte(schema.orders.amount, 1000)
  ))
// result.count: number
```

## Count by Group

```typescript
import { count } from 'drizzle-orm'

const statusCounts = await db.select({
  status: schema.orders.status,
  count: count(),
}).from(schema.orders)
  .groupBy(schema.orders.status)
// [{ status: 'active', count: 42 }, { status: 'pending', count: 15 }]
```
