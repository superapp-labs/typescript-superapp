---
title: update
description: Update one or more rows matching a condition.
---

`update` modifies rows matching the `where` condition and returns the updated records.

```typescript
import { createClient } from '@superapp/db'
import type { SuperAppSchema } from '../generated/schema'

const db = createClient<SuperAppSchema>({ url, userToken })

const updated = await db.main.orders.update({
  where: { id: { $eq: 'ord_abc123' } },
  data: { status: 'shipped' },
})
// updated: { id: string; amount: number; status: string; ... }[]
```

## Options

| Option | Type | Required | Description |
| --- | --- | --- | --- |
| `where` | `object` | Yes | Filter conditions for rows to update |
| `data` | `object` | Yes | Column-value pairs to set |

## Single Row Update

```typescript
const updated = await db.main.orders.update({
  where: { id: { $eq: 'ord_abc123' } },
  data: {
    status: 'delivered',
    amount: 300,
  },
})
// updated: { id: string; amount: number; status: string; ... }[]
```

## Bulk Update

Update multiple rows by using a broader filter condition.

```typescript
const updated = await db.main.orders.update({
  where: { status: { $eq: 'pending' } },
  data: { status: 'cancelled' },
})
// updated: { id: string; amount: number; status: string; ... }[]
// All previously pending orders are now cancelled
```

## Update with Complex Filters

```typescript
const updated = await db.main.orders.update({
  where: {
    status: { $eq: 'pending' },
    created_at: { $lt: '2024-01-01' },
  },
  data: { status: 'expired' },
})
// updated: { id: string; amount: number; status: string; ... }[]
```

## Update a Customer

```typescript
const updated = await db.main.customers.update({
  where: { id: { $eq: 'cust_abc123' } },
  data: {
    name: 'Acme Corporation',
    email: 'new-billing@acme.com',
  },
})
// updated: { id: string; name: string; email: string; ... }[]
```
