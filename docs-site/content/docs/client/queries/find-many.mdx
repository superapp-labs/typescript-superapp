---
title: findMany
description: Fetch multiple rows with filtering, sorting, pagination, and relation includes.
---

`findMany` retrieves multiple rows from a table with optional filtering, sorting, and pagination.

```typescript
import { createClient } from '@superapp/db'
import type { SuperAppSchema } from '../generated/schema'

const db = createClient<SuperAppSchema>({ url, userToken })

const orders = await db.main.orders.findMany({
  select: ['id', 'amount', 'status'],
  limit: 10,
})
// orders: { id: string; amount: number; status: string }[]
```

## Options

| Option | Type | Required | Description |
| --- | --- | --- | --- |
| `select` | `string[]` | No | Columns to return. Omit to return all permitted columns |
| `where` | `object` | No | Filter conditions. See [Filter Operators](/docs/client/queries/filtering) |
| `include` | `object` | No | Related tables to include |
| `orderBy` | `object` | No | Sort order. `{ column: 'asc' \| 'desc' }` |
| `limit` | `number` | No | Maximum number of rows to return |
| `offset` | `number` | No | Number of rows to skip |

## With Filters

```typescript
const activeOrders = await db.main.orders.findMany({
  select: ['id', 'amount', 'status', 'created_at'],
  where: {
    status: { $eq: 'active' },
    amount: { $gte: 100 },
  },
})
// activeOrders: { id: string; amount: number; status: string; created_at: string }[]
```

## With Includes

Load related data in a single query.

```typescript
const ordersWithCustomer = await db.main.orders.findMany({
  select: ['id', 'amount', 'status'],
  include: {
    customer: {
      select: ['id', 'name', 'email'],
    },
  },
})
// ordersWithCustomer: {
//   id: string; amount: number; status: string;
//   customer: { id: string; name: string; email: string }
// }[]
```

## With Sorting and Pagination

```typescript
const page2 = await db.main.orders.findMany({
  select: ['id', 'amount', 'created_at'],
  orderBy: { created_at: 'desc' },
  limit: 20,
  offset: 20,
})
// page2: { id: string; amount: number; created_at: string }[]
```

## Complex Filters

Combine conditions with `$and`, `$or`, and `$not`.

```typescript
const filtered = await db.main.orders.findMany({
  select: ['id', 'amount', 'status'],
  where: {
    $or: [
      { status: { $eq: 'active' }, amount: { $gte: 500 } },
      { status: { $eq: 'pending' }, amount: { $lt: 100 } },
    ],
  },
})
// filtered: { id: string; amount: number; status: string }[]
```

## Select All Columns

Omit `select` to return all columns the user has permission to access.

```typescript
const allColumns = await db.main.orders.findMany({
  where: { status: { $eq: 'active' } },
  limit: 10,
})
```
