---
title: Find First
description: Fetch a single record with optional relation loading.
icon: Target
---

Use `db.query.*.findFirst()` to retrieve a single record. Returns `undefined` if no match.

```typescript
import { drizzle } from '@superapp/db'
import { eq } from 'drizzle-orm'
import * as schema from './generated/schema'

const db = drizzle({ connection, token, schema })

const order = await db.query.orders.findFirst({
  where: eq(schema.orders.id, 'ord_abc123'),
})
// order: { id, amount, status, ... } | undefined
```

## With Relations

```typescript
const order = await db.query.orders.findFirst({
  where: eq(schema.orders.id, 'ord_abc123'),
  with: {
    customer: true,
  },
})
// order: { id, amount, status, customer: { id, name, email } } | undefined
```

## With Select Builder

Alternatively, use the select builder with `.limit(1)`:

```typescript
const [order] = await db.select()
  .from(schema.orders)
  .where(eq(schema.orders.id, 'ord_abc123'))
  .limit(1)
// order: { id, amount, status, ... } | undefined
```

## Handling Undefined Results

```typescript
const order = await db.query.orders.findFirst({
  where: eq(schema.orders.id, 'ord_abc123'),
})

if (!order) {
  console.log('Order not found')
  return
}

console.log(order.amount)
```
